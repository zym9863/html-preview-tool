# HTML预览工具

[English](README_EN.md)

一个简单而强大的HTML实时预览工具，支持实时编辑和预览HTML代码。

## 功能特点

- 实时HTML代码预览
- 支持语法高亮
- 响应式设计，支持移动设备
- 简洁美观的用户界面
- 基于CodeMirror的代码编辑器

## 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/zym9863/html-preview-tool.git
```

2. 打开项目目录：
```bash
cd html-preview-tool
```

3. 在浏览器中打开`index.html`文件即可使用

## 使用方法

1. 在左侧编辑器中输入HTML代码
2. 右侧窗口会实时显示预览效果
3. 支持响应式设计，可在移动设备上使用

## 技术栈

- HTML5
- CSS3
- JavaScript
- CodeMirror编辑器
